Configure GitHub

Create a GitHub Repository

To conveniently deploy our personal website, we will use the service provided by GitHub Pages.
First, you need a GitHub account; if you don‚Äôt have one, please register using your email.

Log into your GitHub account and create a new repository. The repository name must be [your_name.github.io].

Configure GitHub Connection

In order to update our website on GitHub, we need to configure our computer to connect to GitHub. There are two main options for doing this:

Configure using GitHub Desktop

This method automatically connects your computer to your GitHub account without needing to configure SSH keys manually, saving time but may occur some issues. If failed, please configure using SSH Key.

  1. Download and install GitHub Desktop.
  2. Open GitHub Desktop and log in with your GitHub account credentials.
  3. Once logged in, GitHub Desktop will automatically bind your computer to your GitHub account.

Configure using SSH Key (recommended)

This method requires you to generate an SSH key and add it to your GitHub account. This method is more secure and recommended.

  • For Windows Users:
  1. Open Git Bash in your user directory(e.g., C:\Users\YourUsername).
  2. Enter cd ~/.ssh.
    • If you successfully enter the ./ssh folder, skip to step 5. Otherwise, the SSH key does not exist, and you should follow the steps to configure it.
  • For Mac Users:
  1. Open Command Line and enter
    ls -al ~/.ssh.
  2. If a .pub file already exists, you can open it directly using the following command:
    cat ~/.ssh/id_rsa.pub.
    skip to step 5.
