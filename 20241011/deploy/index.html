Build Your Blog (2) - Deploy

Created2024-10-11|Updated2024-10-16|Build Your Blog
|Post Views:

In this step, we will deploy your local personal blog to the internet. We will use GitHub Pages, as it offers a free domain and automatically updates your website whenever you push changes.

Step 1: Check GitHub Pages

First, open the repository you created on Github in the previous step, named [your_name.github.io]. On the repository page, click on Settings, then click Pages. Here, you will see that GitHub Pages has automatically deployed the content of the repository to https://[your_name].github.io/, which will be your website’s home address.

If your page displays “Your site is live at https://[your_name].github.io/,” it means you have successfully set up GitHub Pages.
