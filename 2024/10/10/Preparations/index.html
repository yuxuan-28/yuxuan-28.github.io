Step 1 : Initial Preparations

In this guide, we will walk through the first step in creating your personal website—setting up the development environment. This includes installing necessary tools like Node.js, Git, and Hexo, a popular static site generator. Let’s get started!

Node.js is a JavaScript runtime that allows you to run JavaScript outside of a browser. It’s essential for managing dependencies and running the Hexo framework.

Configure GitHub

Create a GitHub Repository

To conveniently deploy our personal website, we will use the service provided by GitHub Pages.
First, you need a GitHub account; if you don’t have one, please register using your email.

Log into your GitHub account and create a new repository. The repository name must be [your_name.github.io].

Configure GitHub SSH Key

For Windows Users:

  1. Open Git Bash in your user directory(e.g., C:\Users\YourUsername).
  2. Enter cd ~/.ssh.
    • If you successfully enter the ./ssh folder, skip to “Add SSH Key to GitHub”. Otherwise, the SSH key does not exist, and you should follow the steps to configure it.

For Mac Users:

  1. Open Command Line and enter
    ls -al ~/.ssh.
  2. If a .pub file already exists, you can open it directly using the following command:
    cat ~/.ssh/id_rsa.pub.
    skip to “Add SSH Key to GitHub”

    • After execution, it will generate two files, id_rsa and id_rsa.pub, in the .ssh directory (e.g., C:\Users\YourUsername\.ssh).
    • If the following content appears, it indicates that the generation was successful.

Add SSH Key to GitHub

  1. Open the id_rsa.pub file and copy its contents.
  2. Go to your GitHub account, select Settings.
  3. In the Settings page, click on SSH and GPG keys.
  4. You will see New SSH key; click on it.
  5. In the form that appears:
    • Choose a name for the Title.
    • In the Key text area, paste the content you copied from id_rsa.pub.
  6. Finally, click Add SSH key to complete the operation.
